================================================== PANDUAN NAVIGASI ALUR ALAT DATs COLLECTIBLE 001 ================================================== [1] STRUKTUR MENU INDUK UTAMA (PADA LAYAR STANDBY 'DATs Core') -------------------------------------------------- Untuk melompat masuk ke dalam menu program, tekan tombol sistem berikut: * Tombol [ A ] -> Meluncurkan BRANCH 1 (Mode Visual Generatif) [cite: 113-114] * Tombol [ B ] -> Meluncurkan BRANCH 2 (Retro 9-Games Engine) [cite: 114-115] * Tombol [ C ] -> Membuka Pintu Akses MODE C (Web Serial Controller) * Tombol [ D ] -> Membuka Pintu Akses MODE D (TouchDesigner UDP Link) [cite: 111-112] [2] CARA MEMILIH SUB-MODE VISUAL / GAME (ANGKA 1 S/D 9) -------------------------------------------------- Ketika Anda berada di dalam Mode Visual atau Mode Game, Anda dapat berganti sub-mode isi dengan menekan tombol angka murni di bawah ini [cite: 115-116]: * Tekan tombol [ 1 ] -> Membuka Sub-Mode 1 * Tekan tombol [ 7 ] -> Membuka Sub-Mode 2 * Tekan tombol [ 4 ] -> Membuka Sub-Mode 3 * Tekan tombol [ 2 ] -> Membuka Sub-Mode 4 * Tekan tombol [ 8 ] -> Membuka Sub-Mode 5 * Tekan tombol [ 5 ] -> Membuka Sub-Mode 6 * Tekan tombol [ 3 ] -> Membuka Sub-Mode 7 * Tekan tombol [ 9 ] -> Membuka Sub-Mode 8 * Tekan tombol [ 6 ] -> Membuka Sub-Mode 9 [3] PROSEDUR KONFIRMASI (KHUSUS MODE C DAN D) -------------------------------------------------- Saat menekan tombol C atau D di menu awal, sistem tidak langsung masuk demi keamanan data. Layar OLED simulator akan berkedip memunculkan pesan "LAUNCH MODE C/D?". -> Tekan tombol [ * ] untuk memberikan konfirmasi eksekusi (YES / CONFIRM) [cite: 105-108]. [4] TOMBOL RESET GLOBAL / AMAN -------------------------------------------------- Tekan tombol [ 0 ] kapan saja untuk membatalkan proses, keluar dari game/visual aktif, dan kembali ke halaman utama boot awal.
